# Input

The program input is the resume information that is written into the React component. This includes the student name, contact information, education, skills, work experience, and projects. The user does not type information into a form. Instead, the input is provided by the developer through the content inside `Resume.js`. The browser also receives the React files, CSS file, and supporting project files when the application runs.

# Process

The program processes the resume information by using React to organize it into reusable interface sections. `App.js` imports the `Resume` component and displays it inside the main application. `Resume.js` structures the information using JSX, including headings, paragraphs, lists, and links. `Resume.css` applies formatting rules such as font, spacing, colors, layout, and background color. React then renders the component structure into the browser as a single-page web application.

# Output

The output is a formatted resume webpage displayed in the browser. The page shows the student’s contact information, education, skills, work experience, and projects in a clean layout.
